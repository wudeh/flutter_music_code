## 说明

- 数据来源：https://github.com/Binaryify/NeteaseCloudMusicApi
- Vue3 版：https://github.com/wudeh/neteasecloudmusic
- 说明：实现主要的音乐播放功能，其他有生之年再搞，也可能鸽了

## 功能
- 音乐播放，缓存播放，查看评论
- 歌词滚动，手动滑动歌词进度跳转
- 下载：暂停，继续，清空下载（包括本地文件）
- 搜索单曲，歌单
- 更新安装文件
- 网易云登录（暂时只支持验证码登录）
- 私信（未完成）
